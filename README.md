## Getting start

### プレビューモード

```sh
pnpm dev
```

### 記事の作成

```sh
pnpm new
```

## Setup

### mise

```sh
brew install mise
mise trust
mise install
```

### pnpm

```sh
pnpm i
```

### lefthook

```sh
pnpm prepare
```

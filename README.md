## Getting start

### プレビューモード

```sh
pnpm run preview
```

👀 Preview on http://localhost:8000

### 記事の作成

```sh
pnpm run article
```

記事の URL の一部となる slug を指定して作成できます。

```sh
pnpm run article:slug article-slug
```

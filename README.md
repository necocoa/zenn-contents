## Getting start

### プレビューモード

```sh
yarn preview
```

👀 Preview on http://localhost:8000

### 記事の作成

```sh
yarn article
```

記事の URL の一部となる slug を指定して作成できます。

```sh
yarn article:slug article-slug
```
